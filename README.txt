Trent's Lawn Care website

Open index.html in any modern browser.

Working features:
- Responsive mobile and desktop layout
- Mobile menu
- Owner photo and Louisville-themed hero design
- Tap-to-call and tap-to-text links
- Quote form that opens a pre-filled text message
- Actual Google Maps embed centered on Louisville, Kentucky
- Live YouTube uploads playlist
- Instagram, TikTok, YouTube, and Facebook links
- Expandable customer policy sections

Publishing:
The site is static HTML, so it can be uploaded to Netlify, GitHub Pages, or Cloudflare Pages.
Keep index.html at the root of the published folder.

Privacy:
The website map shows Louisville as the service area and does not publish a home address.
